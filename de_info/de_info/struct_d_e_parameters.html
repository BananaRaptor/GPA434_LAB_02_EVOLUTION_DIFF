Description détaillée

Structure représentant les paramètres d'un algorithme à évolution différentielle.

Cette structure encapsule tous les paramètres nécessaires pour configurer et exécuter l'algorithme à évolution différentielle. Elle offre des fonctionnalités pour définir, accéder et réinitialiser ces paramètres.

◆ isReady()

bool DEParameters::isReady ( ) const

Accesseur vérifiant si les paramètres sont prêts pour être utilisés dans l'algorithme.

Cette fonction retourne vrai seulement si tous les paramètres permettent d'exécuter une simulation. Évidemment, une validation rudimentaire des paramètres est effectués, la convergence des paramètres n'est pas validée et garantie.

Les conditions sont les suivantes :

  • le domaine est défini
  • la taille de la population se retrouve dans l'intervalle [minimum, maximum] défini
  • le facteur de différentiation F se retrouve dans l'intervalle [minimum, maximum] défini
  • le taux de mutation CR se retrouve dans l'intervalle [minimum, maximum] défini
  • le nombre de génération maximal se retrouve dans l'intervalle [minimum, maximum] défini
  • la fonction objective est définie
  • la fonction de fitness est définie
Renvoie
Vrai si les paramètres sont correctement configurés, sinon faux.

// params...
bool readyStatus{ params.isReady() };
