Description détaillée

Classe représentant une solution de l'algorithme à évolution différentielle.

Cette classe gère une solution composée de valeurs réelles pour l'algorithme à évolution différentielle DE.

Elle fournit des fonctionnalités pour évaluer, manipuler et représenter la solution.

De plus, elle stock les valeurs représentant sa performance issu du calcul de la fonction objective et de la fonction de fitness.

◆ toString()

std::string DESolution::toString ( ) const

Convertit la solution en chaîne de caractères.

Renvoie
Une représentation textuelle de la solution. La présentation doit être sous ce format : [value1, value2, value3]
  • objective value : value
  • fitness value : value

Dans tous les cas, on affiche les valeurs numériques avec une notation fixe avec la précision spécifiée par smFixedNotationPrecision.

double rectanglePerimeter(DESolution const& solution) {
double width{ solution[0] };
double height{ solution[1] };
return 2.0 * width + 2.0 * height;
}
DEDomain domain;
domain.resize(2);
domain.setAllIntervals(0.0, 100.0);
DESolution solution(domain.size());
solution[0] = 50.0;
solution[1] = 25.0;
solution.evaluate(&rectanglePerimeter, DEFitness::identity);
DESolution::setFixedNotationPrecision(4);
std::cout << solution.toString();
// La sortie est :
// [50.0000, 25.0000]
// - objective value : 150.0000
// - fitness value : 150.0000
