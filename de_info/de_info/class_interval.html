Description détaillée

Classe représentant un intervalle entre deux bornes.

Cette classe gère un intervalle entre deux nombres réels, avec des fonctionnalités pour vérifier si un nombre est contenu dans l'intervalle et pour obtenir une valeur aléatoire dans l'intervalle.

La classe valide en tout temps la validité de l'intervalle. Les règles suivantes s'appliquent :

\(lower \leq higher\)

\(lower \leq value \leq higher\)
