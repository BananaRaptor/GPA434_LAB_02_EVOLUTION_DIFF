◆ FitnessFunction

FitnessFunction

#include <DEFunctions.h>

Définit le type de fonction pour l'évaluation de la fitness d'une solution.

Ce type permet d'identifier la signature d'une fonction pouvant être utilisée pour évaluer la "fitness" (adaptabilité) d'une solution. Elle retourne une valeur de type double qui représente la mesure de performance/d'adaptabilité de la solution.

L'objectif de cette fonction est de normaliser la valeur de la fonction objective afin de garantir une valeur de maximisation.

Paramètres
valueLa valeur objective à évaluer.
Renvoie
La valeur de fitness de la solution.
double identityFunction(double value) {
return value;
}
FitnessFunction fitFunc{ identityFunction };
