Description détaillée

Classe implémentant l'algorithme d'évolution différentielle.

Cette classe fournit une implémentation complète de l'algorithme d'évolution différentielle. Elle permet de configurer, d'exécuter et de récupérer les résultats de l'algorithme à partir d'un ensemble donné de paramètres.
