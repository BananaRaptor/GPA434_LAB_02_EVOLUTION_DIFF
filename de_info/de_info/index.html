Introduction

Cette documentation présente l'ensemble des classes nécessaire à une implémentation orienté objet de l'algorithme à évolution différentielle.

L'implémentation est basée sur la classe principale :

  • DifferentialEvolution

Cette class dépend de :

  • ces classes utilitaires :
    • DEDomain
    • DEParameters
    • DEPopulation
    • DESolution
    • DEStatistics
    • Interval
  • ces alias de types :
    • ObjectiveFunction
    • FitnessFunction

Cette documentation couvre la conception attendue et l'implémentation à réaliser.
